# Dots and Boxes 
 
A peer-to-peer multiplayer Dots and Boxes game that works in any modern web browser. 
 
## Features 
 
- **Local Mode**: Play on the same device (pass and play) 
- **Online Mode**: Connect with friends using simple 6-character room codes 
- **Responsive Design**: Works on desktop and mobile devices 
- **Configurable Grid**: Choose from 3x3, 5x5, or 7x7 grids 
- **Session Stats**: Track wins across multiple games 
 
## How to Play 
 
1. Choose between Local Mode or Online Mode 
2. Select your preferred grid size 
3. Take turns drawing lines between dots 
4. Complete a box to score a point and take another turn 
5. Player with the most boxes wins! 
 
## Technology 
 
- Vanilla JavaScript (ES6 modules) 
- HTML5 Canvas for rendering 
- PeerJS for P2P connections 
- CSS3 for responsive design 
 
## Development 
 
Simply open `index.html` in a web browser to run locally. 
 
For online mode testing, you'll need to: 
1. Host the files on a web server (GitHub Pages, local server, etc.) 
2. Open in two separate browsers/devices 
3. One player hosts, shares the room code 
4. Other player joins using the room code 
 
## Deployment 
 
This project is configured for GitHub Pages deployment. 
Simply push to your repository and enable GitHub Pages in settings.
